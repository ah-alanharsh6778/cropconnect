# Project-CropConnect 
Java Full Stack Developer | Spring Boot | React

## Overview
CropConnect is a full-stack web application that connects farmers and buyers,
allowing farmers to list products and buyers to place orders.

## Tech Stack
- Backend: Java, Spring Boot, Spring Security, JWT
- Frontend: React.js, JavaScript, HTML, CSS
- Database: MySQL
- Tools: Git, GitHub

## Features
- User authentication (Farmer / Buyer)
- Product listing & order management
- Role-based access control
- Secure REST APIs

- crop-backend/
